!DOCTYPE
It informs the browser about the type and version of the document.
html
This is the root element of an HTML document. It encompasses all the content and other HTML elements.
title
Sets the title of the document, which appears in the browser tab.
meta
Provides metadata such as the character set, author, and viewport settings.
link
Links to external resources like CSS files.
Body
This section contains the actual content of the website that is displayed by the browser.
h1
Largest heading
h2
Second largest heading
h3
Third largest heading
h4
Fouth largest heading
h5
fifth largest or second smallest heading
h6
Smallest heading
pre
This tag is used for preformatted text. It preserves spaces, line breaks, and formatting as they are in the HTML code.
p
This tag is used to define a paragraph. It is a block-level element and automatically adds some space before and after the content.
table
The container for the table.
tr(table row)
Defines a row in the table.
th(table header)
Defines a header cell in the table.
td(table data)
Defines a standard cell in the table.
border
Defines the border width of the table and its cells.
cellpadding
Defines the space between the cell content and its border.
cellspacing
Defines the space between cells.
width and height
Defines the width and height of the table.
bgcolor
Defines the background color of the table, rows, or cells.
thead
Groups the header content in a table.
tbody
Groups the body content in a table.
tfoot
Groups the footer content in a table.
caption
A caption tag is used to provide a title for the table.

converts the text to italic text
em
o indicate semantic importance and emphasis, respectively. They provide meaning to the text rather than just changing its appearance. Screen readers and search engines interpret this tag differently
u
The u tag is used to underline text. However, it's generally not recommended because underlined text is commonly associated with hyperlinks.
del
Indicates deleted text, typically rendered with a strikethrough
mark
Highlights text, often used to mark text as important
sup
tag is used to display superscript.They are useful for mathematical equations, footnotes, and chemical formulas.
sub
tags are used to display superscript text, respectively. They are useful for mathematical equations, footnotes, and chemical formulas.
abbr
tags are used to define abbreviations, respectively. They can improve accessibility by providing the full expansion of the abbreviation when hovered over
blockquote
Defines a section that is quoted from another source.
q
it is used to insert quotations.
cite
Defines the title of a work, such as a book, article, or song.
address
Provides contact information, usually rendered in italics.
code
The code tag is used to represent a fragment of computer code. It's typically rendered in a monospaced font and often styled with a background color or border to distinguish it from regular text.
hr
The hr tag is used to create a horizontal line or thematic break in the content. It's often used to separate sections of a webpage.
br
The br tag is used to create a line break within a paragraph. It should be used sparingly, and CSS (margin and padding) should be preferred for controlling spacing between elements.
a(anchor)
It has an opening tag and a closing tag. The content that goes between these tags becomes the clickable part of the link.
href attribute
The most important attribute of an anchor tag is href, which stands for "hypertext reference". It specifies the URL of the page the link goes to.
text link attribute
The text between the opening and closing tags is what the user sees and clicks on. This is called the link text or anchor text.
target attribute
he target attribute specifies where to open the linked document. It can take values like _blank (opens the linked document in a new window or tab) or _self (opens the linked document in the same frame or tab).
img
The img tag is used to embed images into an HTML document.
src attribute
Specifies the URL or path to the image file. This attribute is required.
alt attribute
Provides alternative text for the image, which is displayed if the image cannot be loaded or by screen readers for accessibility and used to describe images to visually impaired users.This attribute is recommended for all img tags.
width attribute
Specifies the width of the image in pixels or as a percentage of the containing element.
height attribute
Specifies the height of the image in pixels or as a percentage of the containing element.
title attribute
Adds a tooltip that appears when the user hovers over the image.
absolute path
An absolute path specifies the complete URL to a resource. This means it includes the protocol (such as http:// or https://), the domain name, and the full path to the resource.
relative path
A relative path specifies the location of a resource in relation to the current document. Relative paths are shorter and more flexible than absolute paths.
